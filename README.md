### 요구사항

- 필요한 데이터를 모두 TypeScript's Interface로 모델링한다.
- 사용되는 모든 함수를 선언부만 만든다.
- 함수 및 클래스의 내부는 구현하지 않습니다.
- Interface만을 따라 작성한 코드가 동작하는 애플리케이션이 되도록 유도하셔야합니다.
